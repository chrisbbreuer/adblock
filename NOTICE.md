# Notices

Adblock v0.1.0 ships a small curated seed ruleset maintained in this repository.

The initial static rules are inspired by common public ad/tracker host patterns used across community-maintained lists such as EasyList and AdGuard filters, but this repository does not vendor those lists directly in v0.1. Future releases should pin upstream list revisions and include exact source URLs, licenses, and generation metadata when broader list ingestion is added.
